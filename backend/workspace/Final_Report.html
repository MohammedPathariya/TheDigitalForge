Implementation Overview

The calculate_factorial function has been implemented with the following key features: - Accepts a non-negative integer as input. - Returns 1 for the edge case where the input is 0. - Raises a ValueError for negative integer inputs, ensuring the function handles invalid cases appropriately.

The implementation is structured in a Python file named math_utils.py, and it is succinct, clear, and efficient.

Final Code Implementation

# math_utils.py

def factorial(n: int) -> int:
    """Calculate the factorial of a non-negative integer n."""
    if n < 0:
        raise ValueError("Factorial is not defined for negative numbers")
    if n == 0 or n == 1:
        return 1
    result = 1
    for i in range(2, n + 1):
        result *= i
    return result
